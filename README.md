# Ruby Lizard

## Set up the dependencies

```bash
$ npm install
$ bundle install
```

## Build the assets

```bash
$ npm run build
```

## Run it!

```bash
$ npm run start
```

## You can set up some Things running:

```bash
$ curl -i -d "name=test&description=testesttest" http://localhot:9293/api/things
```
